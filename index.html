Resumen de la clase

HTML

Es un lenguaje de marcado de hipertexto utilizado principalmente para el desarollo de páginas web, en este se define la estructura del contenido dentro de una página, no le da un diseño o función.

Hipertexto

Se trata de la relación entre páginas ya sea dentro de un sitio o entre varios de ellos. El internet se compone de millones de páginas web y es llamada la W3 (world-wide-web) por esta propiedad que tiene, en la que todo está conectado.

Marcado

El lenguaje HTML utiliza marcado para diferenciar entre texto, imágenes y otro contenido que se muestra dentro del navegador, estos se identifican con etiquetas (tags) como lo pueden ser <p>, <div>, <img>, entre otros.
